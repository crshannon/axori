# Design Tokens

## Colors

### Primary
- `blue-600` - Primary actions
- `blue-700` - Primary hover

### Semantic
- `green-600/700` - Success
- `red-600/700` - Danger/Error
- `yellow-600/700` - Warning
- `gray-200/300` - Secondary

### Neutral
- `gray-50` - Backgrounds
- `gray-100` - Subtle backgrounds
- `gray-200` - Borders
- `gray-500` - Placeholder text
- `gray-700` - Body text
- `gray-900` - Headings

## Typography

### Sizes
- `text-sm` - 14px (Small)
- `text-base` - 16px (Medium/Default)
- `text-lg` - 18px (Large)

### Weights
- `font-normal` - 400
- `font-medium` - 500
- `font-semibold` - 600
- `font-bold` - 700

## Spacing

Use Tailwind's spacing scale:
- `p-2` - 8px
- `p-4` - 16px
- `p-6` - 24px
- `gap-4` - 16px
- `gap-6` - 24px

## Border Radius

- `rounded-md` - 6px (Default)
- `rounded-lg` - 8px (Cards)
- `rounded-full` - 9999px (Badges, Pills)

## Shadows

- `shadow-sm` - Subtle elevation
- `shadow` - Default elevation
- `shadow-lg` - High elevation (Modals)
