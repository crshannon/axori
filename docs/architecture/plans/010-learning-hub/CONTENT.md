## Content Style Guide

### Tone
- **Accessible but not dumbed down** - Respect the reader's intelligence
- **Practical over theoretical** - Focus on "how to use this"
- **Encouraging** - Help readers feel capable of mastering concepts

### Structure for Glossary Terms
1. **Short definition** (1-2 sentences) - Quick reference
2. **Full explanation** - Comprehensive but scannable
3. **Why it matters** - Practical importance
4. **Example** - Real numbers, real scenarios
5. **Formula** (if applicable) - With variable explanations
6. **Pro tips** - Expert-level insights
7. **Related terms** - Build connections

### Writing Guidelines
- Use second person ("you") for engagement
- Avoid jargon without explanation
- Include real numbers in examples
- Link to related concepts liberally
- End with actionable takeaway
